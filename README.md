# Gestión de Alquileres

Una hermosa aplicación web para gestionar tus propiedades de alquiler, construida con tecnologías web modernas y un diseño glassmorphism.

## Características

- **Diseño Premium**: Interfaz moderna con efectos de vidrio y animaciones suaves.
- **Gestión de Propiedades**: Agrega y visualiza tus propiedades con detalles como precio, inquilino y contratos.
- **Finanzas**: Visualiza tus ingresos mensuales totales y desglosados.
- **Responsive**: Funciona perfectamente en móviles y escritorio.
- **Sin Backend**: Todos los datos se guardan localmente en tu navegador.

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador web favorito (Chrome, Edge, Firefox).
2. En la pantalla de login, ingresa cualquier usuario y contraseña para entrar.
3. Explora las secciones:
    - **Inicio**: Resumen de tus propiedades e ingresos.
    - **Propiedades**: Lista de tus inmuebles. Usa el botón `+` para agregar uno nuevo.
    - **Finanzas**: Detalle de tus ingresos.

## Tecnologías

- HTML5
- CSS3 (Variables, Flexbox, Grid, Glassmorphism)
- JavaScript (ES6+, LocalStorage)
